О проекте

Проект был создан в рамках курсовой работы по дисциплине "Кроссплатформенная среда исполнения программного обеспечения".

Основная цель проекта заключалась в изучении и применении Spring и его составляющих для разработки веб-приложений.

Проект включает в себя следующие технологии и инструменты:

  • Spring Framework
  • Spring Security
  • Spring Boot
  • Spring Data
  • Thymeleaf (шаблонизатор)
  • Bootstrap
  • Redis
  • База данных PostgreSQL
  • Инструмент сборки проекта Maven
